## Overview

This is a template for recreating the data pull and calculations used for Comprehensive Plans.  

## Data Prep  

Data is sourced from three different Census products: the 2000 Decennial Census table SF3, variables from the 2000, 2010, and 2020 Decennial Census table SF1, and the American Community Survey 5 Year Estimates.  

From the *2000 Decennial SF3*  
+ Variables P087001 - 2: Poverty by Sex by Age  
+ H03 Series: Units in Structure  
+ Variable HCT012001: Median Household Income  
+ H094 Series: Homeowner Cost Burden  
+ H069 Series: Renter Cost Burden  
+ Variable H076001: Median Value Owner Occupied Home  
+ Variable H063001: Median Gross Rent Renter Occupied  
+ H034 Series: Year Structure Built  
+ Variable H035001: Median Year Structure Built  
+ P03 Series: Commute Mode  
+ H04 Series: Vehicles by Household Tenure  
+ PCT019 Series: Place of Origin for Foreign-Born Population  
+ P019 Series: Language Spoken at Home Population 5+  
+ P037 Series: Educational Attainment by Sex  
+ P0510 Series: Class of Worker  
+ PCT035 Series: Labor Force Status for the Civilian Population by Sex  
+ P049 Series: Industry Employment  
+ P050 Series: Occupational Employment
